Инструкция пользователя

На этой странице собран краткий порядок работы с корпусом для исследователя, редактора и администратора.

Как работать с корпусом

  1. Откройте раздел «Поиск» и задайте текстовый запрос или фильтры по метаданным.
  2. Выберите режим поиска: полнотекстовый, по словоформе, по лемме или по фразе.
  3. При необходимости сузьте выдачу по году, тому, номеру, разделу, автору, языку и ключевым словам.
  4. Из результатов переходите в карточку статьи, просматривайте аннотацию, метаданные, файлы и очищенный текст.
  5. После входа в систему сохраняйте запросы и формируйте подкорпуса для повторного анализа.
  6. Используйте раздел «Аналитика» для частотных списков, биграмм и сравнения подкорпусов.

Где загружать статью

В пользовательской части сайта загрузка доступна через пункт меню «Загрузка статьи». Этот пункт видят пользователи с ролью редактора или администратора.

Если вы вошли под редактором и не видите ссылку, используйте прямой адрес: {% url 'corpus:editor-upload' %}.

Для расширенного редактирования записей, ролей и служебных сущностей доступна также стандартная панель Django admin по адресу /admin/.

Для исследователя

Основные рабочие разделы: поиск, карточки статей, подкорпуса и аналитика.

Для редактора

Используйте форму загрузки статьи для ручного пополнения корпуса и management command синхронизации для автоматического импорта из архива журнала.

Для администратора

Управление ролями, пользователями и служебными сущностями доступно в /admin/.
