# Как работает импорт, поиск и подкорпуса

## Импорт

Система поддерживает два режима пополнения:

1. Ручной ввод через административный интерфейс.
2. Пакетный импорт из каталога с `metadata.json` и файлами.

Во втором случае для каждой статьи создаются:

- выпуск;
- раздел;
- авторы и аффилиации;
- карточка статьи;
- очищенный текст;
- файл публикации;
- токены и леммы для последующего поиска.

## Поиск

- полнотекстовый поиск и поиск по фразе используют `PostgreSQL Full Text Search`;
- поиск по словоформе и лемме использует `ArticleToken` и `Lemma`;
- метаданные фильтруются отдельно и могут сочетаться с любым поисковым режимом.

## Подкорпуса

Подкорпус сохраняет набор фильтров и автоматически формирует состав статей.
Он может использоваться как отдельная база для частотного анализа, биграмм и сравнения с другим подкорпусом.
