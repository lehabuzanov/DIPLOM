# Инструкция пользователя

## Основной сценарий работы

1. Откройте главную страницу корпуса и перейдите в раздел `Поиск`.
2. Задайте текстовый запрос и выберите режим поиска:
   - полнотекстовый;
   - по словоформе;
   - по лемме;
   - по фразе.
3. При необходимости ограничьте выдачу по метаданным:
   - год;
   - том;
   - номер;
   - раздел;
   - автор;
   - язык;
   - ключевые слова.
4. Просмотрите результаты поиска и откройте карточку статьи.
5. В карточке статьи изучите метаданные, аннотацию, ключевые слова, файлы и очищенный текст.
6. После входа в систему сохраните поисковый запрос или сформируйте подкорпус.
7. Перейдите в раздел `Аналитика`, чтобы посмотреть частотный список, биграммы и сравнение двух статей или двух подкорпусов.

## Где искать нужные функции

- `Главная`: краткое описание ресурса, общая статистика и быстрый поиск.
- `Выпуски`: архив выпусков журнала.
- `Статьи`: каталог статей с фильтрацией.
- `Поиск`: основной рабочий режим для корпусного и метаданных поиска.
- `Подкорпуса`: сохранённые выборки пользователя.
- `Аналитика`: частотность, графики и сравнение материалов.

## Сохранение результатов

- Сохранение поисковых запросов доступно после авторизации.
- Подкорпус можно сформировать из текущих фильтров на странице поиска.
- Результаты поиска можно выгрузить в `CSV`.

## Загрузка новой статьи

Для обычного исследователя ручная загрузка статьи не отображается.

Для редактора или администратора в меню доступен пункт `Загрузка статьи`.
Также форму можно открыть напрямую по адресу `/corpus/articles/upload/`.
