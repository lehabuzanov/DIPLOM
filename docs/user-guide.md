# Инструкция пользователя

## Основной сценарий работы

1. Откройте главную страницу корпуса.
2. Перейдите в раздел `Поиск`.
3. Задайте текстовый запрос и при необходимости фильтры:
   - год;
   - том;
   - номер;
   - раздел;
   - автор;
   - язык;
   - ключевое слово;
   - название статьи.
4. Выберите режим поиска:
   - полнотекстовый;
   - по словоформе;
   - по лемме;
   - по фразе.
5. Изучите результаты и откройте карточку статьи.
6. При необходимости сохраните запрос или сформируйте подкорпус.
7. Для статистики и сравнения перейдите в раздел `Аналитика`.

## Различие между разделами `Статьи` и `Поиск`

- `Статьи` — каталог публикаций с фильтрами по метаданным и быстрым обзором корпуса.
- `Поиск` — основной исследовательский режим, где доступны текстовый запрос, поиск по лемме, словоформе, фразе, контексты и сохранение выборок.

## Что есть в карточке статьи

- название;
- авторы;
- год, том, номер, раздел;
- аннотация;
- ключевые слова;
- очищенный текст;
- ссылка на страницу статьи или PDF;
- добавление статьи в подкорпус;
- похожие статьи.

Если локальный файл статьи отсутствует, ссылка автоматически переводит на рабочий внешний файл или страницу журнала.

## Сохраненные запросы

После входа в систему на странице `Поиск` можно:

- сохранить текущий запрос под собственным названием;
- открыть его повторно;
- изменить название и описание;
- удалить его.

Раздел со всеми сохраненными запросами:

```text
/corpus/queries/
```

## Подкорпуса

Подкорпус — это сохраненная выборка статей по заданным критериям.

С подкорпусом можно:

- сохранить текущую выборку из поиска;
- открыть список статей;
- пересобрать состав по исходным фильтрам;
- изменить название и описание;
- удалить подкорпус;
- открыть его в аналитике.

Раздел подкорпусов:

```text
/corpus/subcorpora/
```

## Аналитика

В разделе `Аналитика` доступны:

- частотный список словоформ;
- частотный список лемм;
- биграммы;
- динамика публикаций по годам;
- самые активные авторы;
- сравнение двух статей;
- сравнение двух подкорпусов.

## Регистрация

Регистрация работает через обычную форму сайта.

Пользовательские данные:

- сохраняются в основной базе PostgreSQL;
- остаются в системе после перезапуска сайта;
- доступны в личном кабинете после входа.

## Где находится загрузка статьи

Ручная загрузка статьи доступна только редактору или администратору.

Путь:

```text
/corpus/articles/upload/
```

Обычный исследователь эту вкладку не видит.

## Примеры полезных запросов

- `управление` в режиме `по лемме`
- `перевод` в режиме `полнотекстовый`
- `нежность` в режиме `по лемме`
- статьи раздела лингвистики за 2023–2025 годы
- статьи конкретного автора с ключевым словом `корпус`
