# Инструкция пользователя

## Основной сценарий работы

1. Откройте главную страницу корпуса.
2. Перейдите в раздел `Поиск`.
3. Задайте текстовый запрос и при необходимости фильтры:
   - год;
   - том;
   - номер;
   - автор;
   - язык;
   - ключевое слово;
   - название статьи.
4. Выберите режим поиска:
   - полнотекстовый;
   - по словоформе;
   - по лемме;
   - по фразе.
5. Изучите результаты и откройте карточку статьи.
6. При необходимости сохраните запрос или сформируйте подкорпус.
7. Для статистики и сравнения перейдите в раздел `Аналитика`.

## Различие между разделами `Статьи` и `Поиск`

- `Статьи` — каталог публикаций с фильтрами по метаданным и быстрым обзором корпуса.
- `Поиск` — основной исследовательский режим, где доступны текстовый запрос, поиск по лемме, словоформе, фразе, контексты и сохранение выборок.

## Что есть в карточке статьи

- название;
- авторы;
- год, том и номер;
- аннотация;
- ключевые слова;
- очищенный текст;
- ссылка на страницу статьи или PDF;
- добавление статьи в подкорпус;
- пометки в тексте для зарегистрированного пользователя;
- похожие статьи.

Если локальный файл статьи отсутствует, ссылка автоматически переводит на рабочий внешний файл или страницу журнала.

## Пометки

После входа в систему можно выделить фрагмент в основном тексте статьи и сохранить его как пометку. Все сохраненные фрагменты собираются в личном кабинете в разделе `Пометки`: там статьи раскрываются по одной, а стрелки позволяют быстро переходить между отмеченными местами и открывать нужный фрагмент прямо в тексте. К каждой пометке можно добавить личный комментарий, затем отредактировать его или удалить пометку целиком.

## Сохраненные запросы

После входа в систему на странице `Поиск` можно:

- сохранить текущий запрос под собственным названием;
- открыть его повторно;
- изменить название и описание;
- удалить его.

Раздел со всеми сохраненными запросами:

```text
/corpus/queries/
```

## Подкорпуса

Подкорпус — это сохраненная выборка статей по заданным критериям.

С подкорпусом можно:

- сохранить текущую выборку из поиска;
- открыть список статей;
- пересобрать состав по исходным фильтрам;
- изменить название и описание;
- удалить подкорпус;
- открыть его в аналитике.

Раздел подкорпусов:

```text
/corpus/subcorpora/
```

## Аналитика

В разделе `Аналитика` доступны:

- частотный список словоформ;
- частотный список лемм;
- биграммы;
- динамика публикаций по годам;
- самые активные авторы;
- сравнение двух статей;
- сравнение двух подкорпусов.

## География авторов

Раздел `Карта` показывает города, с которыми связаны аффилиации авторов статей. Можно выбрать год выпуска из реально имеющихся годов или ввести город через поле с подсказками. Для выбранного года страница показывает:

- карту городов с размером точки по числу авторов;
- лидеров года по городам;
- число городов, авторов и статей;
- диаграмму активности городов;
- для выбранного города — динамику по годам и список статей.

После нового импорта выпусков географию можно пересобрать командой:

```text
python manage.py refresh_author_geography
```

Команда использует встроенный справочник распространенных городов и нормализацию названий организаций. Для новых или нестандартных аффилиаций можно запустить режим внешнего геокодирования:

```text
python manage.py refresh_author_geography --geocode
```

Такие результаты помечаются как требующие проверки редактором.

## Регистрация

Регистрация работает через обычную форму сайта.

Пользовательские данные:

- сохраняются в основной базе PostgreSQL;
- остаются в системе после перезапуска сайта;
- доступны в личном кабинете после входа.

После регистрации пользователь получает роль `Исследователь`. Она дает доступ к личным
инструментам: сохраненным запросам, подкорпусам, пометкам и повторному использованию
результатов анализа. Роли редактора и администратора самостоятельно получить нельзя:
их назначает администратор корпуса.

## Смена пароля

В личном кабинете доступна смена пароля без выхода из системы:

```text
/accounts/password/change/
```

Для смены нужно указать текущий пароль и новый пароль, соответствующий правилам Django.

## Где находится загрузка статьи

Ручная загрузка статьи доступна только редактору или администратору.

Путь:

```text
/corpus/articles/upload/
```

Обычный исследователь эту вкладку не видит.

## Примеры полезных запросов

- `управление` в режиме `по лемме`
- `перевод` в режиме `полнотекстовый`
- `нежность` в режиме `по лемме`
- статьи раздела лингвистики за 2023–2025 годы
- статьи конкретного автора с ключевым словом `корпус`
